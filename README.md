# 射龍門
使用 ReactJS & Next.js 開發的小遊戲
快來與朋友線上連線賭一把八～

#### 環境建置
##### 步驟1. 安裝 node14.15.3
```
> cd nextjs-let-it-ride;
> nvm use 18.5.0;
> npm install;
```

##### 步驟2. start docker
```
> cd docker;
> docker-compose up -d;
```

##### 步驟3. 瀏覽
http://localhost/

#### 開發工具
1. next.js 
2. reactJS
2. socket.io
